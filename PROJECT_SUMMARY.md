# Project Summary: Zodu-Website

**Purpose:** A comprehensive Restaurant Management and Point of Sale (POS) system.

**Technology Stack:**
*   **Framework:** A modern single-page application (SPA) built with **React** and **Vite**.
*   **Language:** **TypeScript**.
*   **UI:** Uses **Material-UI (MUI)** for its component library.
*   **State Management:** Employs **Redux Toolkit**, including **RTK Query** for efficient server state management and caching.
*   **API Communication:** Uses **Axios** for making HTTP requests to a backend API. Authentication is handled automatically by attaching a JWT Bearer token to requests.

**Key Features:**
The application is well-structured around core restaurant management functionalities:
*   **Point of Sale (POS):** A dedicated interface for creating customer orders.
*   **Restaurant Setup:** Configuration for restaurant details.
*   **Menu & Inventory Management:** Tools for creating and tracking menu items and stock.
*   **Expense Tracking:** A feature for logging and viewing business expenses.
*   **Reporting:** A section for data overviews and reports.
*   **Authentication:** A secure, token-based login system.

In short, this is a robust and modern web application for managing restaurant operations, built with a scalable and maintainable architecture.
